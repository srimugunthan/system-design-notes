# L14: Fine-Tuning & Distillation for Edge

## Introduction

Adapting LLMs for edge deployment requires fine-tuning and distillation to create efficient, task-specific models. Techniques like LoRA and QLoRA enable parameter-efficient training, while distillation compresses models for resource-constrained devices.

## LoRA (Low-Rank Adaptation)

Efficient fine-tuning with low-rank matrices.

### Theoretical Background

Freeze original weights, train low-rank adapters.

W' = W + BA, where B and A are low-rank matrices.

### Practical Implementation

In PEFT library:

```python
from peft import LoraConfig, get_peft_model

config = LoraConfig(r=8, lora_alpha=32)
model = get_peft_model(model, config)
```

Reduces trainable parameters by 90%.

Reference:
- Paper: Hu, E., et al. (2021). LoRA: Low-Rank Adaptation of Large Language Models. arXiv:2106.09685

## QLoRA

Quantized LoRA for memory efficiency.

### Theoretical Background

Quantize base model to 4-bit, train LoRA adapters.

Combines quantization with PEFT.

### Practical Implementation

In transformers:

```python
from transformers import BitsAndBytesConfig

quant_config = BitsAndBytesConfig(load_in_4bit=True)
model = AutoModelForCausalLM.from_pretrained("model", quantization_config=quant_config)
# Then apply LoRA
```

Enables fine-tuning 65B models on single GPU.

Reference:
- Paper: Dettmers, T., et al. (2023). QLoRA: Efficient Finetuning of Quantized LLMs. arXiv:2305.14314

## Knowledge Distillation

Transfer knowledge from large to small models.

### Theoretical Background

Train small "student" model to mimic large "teacher" model.

Loss: KL divergence between teacher and student logits.

### Practical Implementation

For edge: distill Llama-65B to MobileLlama-1B.

Techniques: response-based distillation, intermediate distillation.

Reference:
- Paper: Hinton, G., et al. (2015). Distilling the Knowledge in a Neural Network. arXiv:1503.02531

## Model-to-Edge Pipeline

End-to-end process for edge deployment.

### Theoretical Background

Fine-tune → Distill → Quantize → Optimize → Deploy

### Practical Implementation

1. Fine-tune with LoRA/QLoRA
2. Distill to smaller model
3. Quantize with GPTQ/AWQ
4. Convert to GGUF/ONNX
5. Deploy with llama.cpp/MLX
