##  System Design Skill Set

# Python based stack:
---

### Core Competencies

**Frontend**
React + TypeScript with HTML/CSS — production-grade, type-safe UIs with component-driven architecture.

**Backend**
Python + FastAPI — high-performance async REST APIs, schema validation via Pydantic, and OpenAPI docs out of the box.

**Data Layer**
- PostgreSQL — relational workloads, ACID compliance, complex joins
- MongoDB — document store for flexible, schema-evolving data
- SQLite — embedded/lightweight persistence
- Redis — caching, rate limiting, pub/sub, session management
- Elasticsearch — full-text search, log analytics, vector search (kNN)

**Cloud**
GCP — compute, storage, managed services, IAM, and deployment pipelines on Google Cloud.

---

### What You Can Build

**1. Enterprise Search & Analytics Platforms**
Full-text and semantic search over large document corpora — Elasticsearch on the query layer, PostgreSQL for structured metadata, FastAPI serving the API, React dashboards for exploration.

**2. Real-Time Fraud & Risk Dashboards**
Event-driven pipelines with Redis for low-latency feature serving, PostgreSQL for audit trails, and React UIs for ops teams — directly aligned with your fraud/AML domain.

**3. AI-Augmented Internal Tools**
RAG-based knowledge assistants, credit risk audit agents, or LLM-powered document Q&A — FastAPI as the agentic backend, Elasticsearch for retrieval, MongoDB for conversation state.

**4. Multi-Tenant SaaS Applications**
Role-based access, isolated data per tenant, REST API contracts, and GCP-managed deployments — suitable for fintech, HR tech, or compliance tooling.
